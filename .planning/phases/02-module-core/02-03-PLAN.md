    **Step 1 — Create `src/main/repositories/PlannerRepository.ts`:**
    Class with constructor taking `Database.Database`. Methods:

    `listForDate(date: string): Task[]`
    - SELECT id, title, notes, date, completed, position, created_at as createdAt FROM tasks WHERE date = ? ORDER BY completed ASC, position ASC, created_at ASC
    - Map `completed` INTEGER (0/1) to boolean
    - Map `created_at` to `createdAt`

    `create(data: { title: string; notes?: string; date: string }): Task`
    - Generate id with `crypto.randomUUID()`
    - Calculate next position: `SELECT COALESCE(MAX(position), -1) + 1 FROM tasks WHERE date = ?`
    - INSERT INTO tasks (id, title, notes, date, completed, position, priority, created_at) VALUES (?, ?, ?, ?, 0, ?, 0, ?)
    - Return the created task via SELECT

    `update(id: string, data: { title?: string; notes?: string; date?: string; completed?: boolean }): void`
    - Build dynamic SET clause for only provided fields
    - Map `completed` boolean to 0/1 integer for SQL
    - UPDATE tasks SET ... WHERE id = ?

    `delete(id: string): void`
    - DELETE FROM tasks WHERE id = ?

    `reorder(ids: string[], date: string): void`
    - Use `db.transaction()` to update position for each id:
    ```typescript
    const stmt = this.db.prepare('UPDATE tasks SET position = ? WHERE id = ? AND date = ?')
    const tx = this.db.transaction((ids: string[]) => {
      ids.forEach((id, index) => stmt.run(index, id, date))
    })
    tx(ids)
    ```

    `getNotes(date: string): string`
    - SELECT content FROM daily_notes WHERE date = ?
    - Return content or empty string if no row

    `saveNotes(date: string, content: string): void`
    - INSERT OR REPLACE INTO daily_notes (date, content, updated_at) VALUES (?, ?, ?)
    - updated_at = new Date().toISOString()

    ```typescript
    export function registerPlannerHandlers(): void {
      const repo = new PlannerRepository(getDb())

      ipcMain.handle('planner:listForDate', (_, date: string) => repo.listForDate(date))
      ipcMain.handle('planner:create', (_, data) => repo.create(data))
      ipcMain.handle('planner:update', (_, data) => {
        const { id, ...rest } = data
        repo.update(id, rest)
      })
      ipcMain.handle('planner:delete', (_, id: string) => repo.delete(id))
      ipcMain.handle('planner:reorder', (_, data) => repo.reorder(data.ids, data.date))
      ipcMain.handle('planner:getNotes', (_, date: string) => repo.getNotes(date))
      ipcMain.handle('planner:saveNotes', (_, data) => repo.saveNotes(data.date, data.content))
    }
    ```
