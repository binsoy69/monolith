    **Step 1 — Extend `src/renderer/expenses/expenses-store.ts`:**
    Add methods:
    ```typescript
    createExpense: async (data: { amount: number; date: string; categoryId: string; walletId: string; notes?: string }) => Promise<void>
    updateExpense: async (id: string, data: { amount: number; date: string; categoryId: string; walletId: string; notes?: string }) => Promise<void>
    deleteExpense: async (id: string) => Promise<void>
    createCategory: async (data: { name: string; color: string }) => Promise<Category | null>
    updateCategory: async (id: string, data: { name?: string; color?: string }) => Promise<void>
    deleteCategory: async (id: string) => Promise<boolean>
    ```
    Key behaviors:
    - `createExpense`: optimistic — add expense to list + deduct from wallet balance in store, then IPC. On error: rollback + toast "Failed to save expense. Changes were not applied."
    - `updateExpense`: optimistic — update expense in list + reverse old wallet deduction + apply new deduction, then IPC. On error: rollback + toast.
    - `deleteExpense`: optimistic — remove from list + reverse wallet deduction, then IPC. On error: rollback + toast.
    - `createCategory`: calls IPC, reloads categories, returns the new category (needed to auto-select in form)
    - `deleteCategory`: calls IPC, returns boolean. If false, toast "Can't delete -- this category is in use"
