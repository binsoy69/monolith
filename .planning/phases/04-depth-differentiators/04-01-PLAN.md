Planned contract for this plan:

```ts
export type HabitKind = 'boolean' | 'count'

export interface Habit {
  id: string
  name: string
  daysOfWeek: string
  archived: boolean
  createdAt: string
  kind: HabitKind
  targetCount: number | null
}

export interface HabitWithToday extends Habit {
  completedToday: boolean
  currentStreak: number
  bestStreak: number
  todayValue: number
}

export interface HabitHistoryPoint {
  date: string
  value: number
  completed: boolean
}
```

Important existing constraints:

- `habit_completions.value INTEGER` already exists and must be reused.
- `HabitRepository.listActive()` already orders by `position ASC, created_at ASC`.
- `TaskList.tsx` shows the proven "sortable active bucket, plain completed bucket" pattern to mirror.
