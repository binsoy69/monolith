## Spacing Scale

Declared values (multiples of 4, locked in `.impeccable.md`):

| Token | Value | Usage |
|-------|-------|-------|
| `--space-1` | 4px | Icon gaps, inline padding, heatmap cell gaps |
| `--space-2` | 8px | Compact element spacing, chart axis padding |
| `--space-4` | 16px | Default element spacing, card padding |
| `--space-6` | 24px | Section padding, expanded card content inset |
| `--space-8` | 32px | Layout gaps between major sections |

Exceptions:
- Heatmap cells: 10×10px with 2px gap (not on 4px grid — visual density exception for the 90-day grid to fit inside expanded HabitCard without scrolling)
- Chart containers: min-height 160px for donut, 120px for trend line

## Component Inventory

### Habit Module — New Components

#### HabitCard Expanded State
- **Trigger:** Click HabitCard → expand inline (same pattern as `expandedTaskId` in PlannerView)
- **Content:** 7-day + 30-day completion counts, 90-day heatmap grid
- **Layout:** Below the card name/streak row, indented `--space-6` from left
- **Transition:** Height auto with `--duration-normal` (150ms)

#### HabitHeatmap
- **Type:** SVG grid, 90 cells (13 columns × 7 rows, partial last column)
- **Cell size:** 10×10px, 2px gap
- **Grid width:** ~156px (fits inside expanded card)
- **Labels:** 3-letter month labels below grid at column start, `--font-size-small`
- **Interaction:** Hover cell → tooltip with date + "Completed" / "Missed"
- **Empty state:** All cells transparent with `1px solid var(--color-border)` outline

#### Count-Based Habit Display
- **Layout:** Fraction text (`3/8`) replaces streak area when habit is count-based
- **Click behavior:** Each click increments count by 1 (same card click handler, no new button)
- **Completed state:** When count ≥ target, card shows same opacity treatment as boolean checked (0.5)
- **Fraction styling:** `--font-size-body`, weight 400, `--color-text-secondary` → `--color-accent` on completion

### Planner Module — Modified Components

#### TaskRow Priority Badge
- **Position:** Inline before task title, 4px gap
- **Size:** Fixed width 24px, height 18px, centered text
- **Border radius:** `--radius-sm` (4px)
- **Typography:** `--font-size-small`, weight 600
- **Visibility:** Only shown when priority is set (default: no priority = no badge)

#### TaskRow Carry-Forward Indicator
- **Position:** 3px left border on the task row
- **Color:** `--color-warning` (`#f59e0b`)
- **Condition:** Task has `carriedFrom` date that differs from current view date
- **Interaction:** None — purely visual indicator

#### TaskRow Overdue Indicator
- **Position:** Small dot or text marker after the task title
- **Color:** `--color-warning` (`#f59e0b`)
- **Condition:** Task date < today AND task not completed
- **Copy:** "overdue" text in `--font-size-small`, `--color-warning`, appended after title

### Expense Module — New Components

#### ExpenseChartSection
- **Position:** Collapsible section above ExpenseList (inside ExpensesView right panel)
- **Toggle:** Text button "Show charts" / "Hide charts" — same styling as "Manage categories" toggle
- **Default state:** Collapsed
- **Layout:** Two charts side by side: donut (left), trend line (right)
- **Container:** `--color-bg-elevated` background, `--radius-md` border radius, `1px solid var(--color-border)`
- **Min height:** 200px when expanded

#### DonutChart
- **Library:** Recharts `<PieChart>` with `<Pie innerRadius={50} outerRadius={70}>`
- **Center content:** Total spending amount in `--font-size-display`, weight 600
- **Legend:** Below chart, horizontal, `--font-size-small` labels with color dots
- **Empty state:** Single gray ring with "No expenses" center text

#### TrendLineChart
- **Library:** Recharts `<AreaChart>` with `<Area type="monotone">`
- **Period toggle:** 3 buttons (3M / 6M / 12M), `--font-size-small`, active state uses `--color-accent`
- **Default period:** 6 months (D-12)
- **Y-axis:** Spending totals, `--font-size-small`, `--color-text-muted`
- **X-axis:** Month abbreviations, `--font-size-small`, `--color-text-muted`
- **Grid lines:** `var(--color-border)` — subtle, not dominant
- **Tooltip:** Dark tooltip matching `--color-bg-overlay`, shows month + total
- **Empty state:** Flat zero line with "No data yet" text

#### MonthlyTotalDisplay
- **Position:** Inside the collapsible chart section header area
- **Content:** "March 2026: ₱12,450" — current month total
- **Typography:** `--font-size-heading`, weight 600

## Accessibility

Per `.impeccable.md` baseline:
- All heatmap cells have `aria-label="{date}: {status}"` for screen readers
- Priority badges use `aria-label="Priority {level}"` not just "P1"
- Carry-forward tasks announce "Carried from {date}" via `sr-only` span
- Chart sections have `role="img"` with `aria-label` describing the data summary
- Period toggle buttons are a `role="tablist"` with `aria-selected`
- Focus rings: 2px solid `--color-accent` on all interactive elements
- `prefers-reduced-motion`: all chart animations and transitions disabled
- Recharts: disable all enter/exit animations when reduced motion is preferred

## Recharts Theme Override

All Recharts components must be styled to match the app's dark, dense aesthetic. No default Recharts styling visible.

```
Recharts overrides:
- Background: transparent (container provides bg)
- Axis tick style: { fill: 'var(--color-text-muted)', fontSize: 12 }
- Axis line: { stroke: 'var(--color-border)' }
- Grid: { stroke: 'var(--color-border)', strokeDasharray: '2 4' }
- Tooltip: { backgroundColor: '#1e1e30', border: '1px solid rgba(255,255,255,0.07)', borderRadius: 6 }
- Tooltip label: { color: 'var(--color-text-primary)', fontSize: 13 }
- Tooltip item: { color: 'var(--color-text-secondary)', fontSize: 12 }
- Cursor: { stroke: 'var(--color-accent)', strokeWidth: 1 }
- No default legend icons — custom color dots (8px circles)
```
