    Export these exact members:

    ```ts
    export interface HabitReminderSnapshot {
      notificationsEnabled: boolean
      notificationTime: string
      lastReminderDate: string | null
    }

    export function shouldSendHabitReminder(
      snapshot: HabitReminderSnapshot,
      now: Date,
      uncheckedCount: number

      stop(): void

    }

    - `shouldSendHabitReminder()` returns `false` when:

      - `lastReminderDate === today`
      - local `HH:mm` is earlier than `notificationTime`
    - `checkNow()` computes unchecked scheduled habits for today using `HabitRepository`
    - when the reminder should fire, create:

      body: `${count} habits unchecked today`,
    })
    ```

    - persist `_lastHabitReminderDate = today`
    - on click:
      - restore the window if minimized
      - focus the window
      - send `mainWindow.webContents.send('shell:navigate', { module: 'habits' })`
